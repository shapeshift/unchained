# Unchained (Go)

## Project Structure

- `pkg` - shared common packages across coinstacks
- `internal` - internal utility packages
- `coinstacks/{coin}` - coin specific logic

## Dependencies

- [Golang](https://go.dev/)

## Initial Setup

- Install [Golang](https://go.dev/doc/install)
- Make coinstacks
  ```sh
  make
  ```
